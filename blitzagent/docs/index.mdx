# Welcome to Blitz <img src="/images/blitzbot.png" alt="BlitzBot" width="50" style={{ display: "inline", marginLeft: "15px", verticalAlign: "middle" }} />

**Blitz** is an AI agent layer for sports that provides developers with powerful tools to build sports-intelligent applications. Our platform starts with the most accurate natural-language sports query agent for NBA, trained on highly processed and optimized historical data. More leagues coming soon.

**Current Capabilities:**
- Natural language sports queries with advanced reasoning
- Comprehensive NBA player and team analysis  
- Real-time data integration and insights
- RESTful API with streaming support

**Platform Vision:**
Our search agent is the foundation for a broader ecosystem of AI agents that automate high-value workflows for sports businesses - from chatbot interfaces and social media content generation to predictive models, scouting reports, and more. The platform integrates with your proprietary data to enable precise, customizable actions for any sports-related need.

**Getting Started:**
Access our current search agent through simple API endpoints, with additional specialized agents coming soon. Combine multiple agents to create sophisticated automated workflows for your sports applications.

**Ready to get started?** Test out our endpoints in the [API Reference](/api-reference/introduction) tab or follow our [Quick Start](/quickstart) guide.

## Example Queries

Here are some examples of what you can ask Blitz:

<CodeGroup>
```bash Player Analysis
curl -X POST "https://blitzagent.onrender.com/analyze" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_BLITZ_API_KEY" \
  -d '{
    "query": "How has LeBron James performed in clutch situations this season?",
    "extra_context": "Focus on games in the last 5 minutes with score within 5 points"
  }'
```

```bash Team Analysis
curl -X POST "https://blitzagent.onrender.com/analyze" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_BLITZ_API_KEY" \
  -d '{
    "query": "Which NBA teams have the best three-point defense this season?",
    "extra_context": "Include opponent shooting percentages and defensive schemes"
  }'
```

```python Python SDK
import httpx
import asyncio

async def analyze_sports():
    async with httpx.AsyncClient() as client:
        response = await client.post(
            "https://blitzagent.onrender.com/analyze",
            headers={"Authorization": "Bearer YOUR_BLITZ_API_KEY"},
            json={
                "query": "Which NBA teams have the best three-point defense?",
                "extra_context": "Include data from the last 20 games"
            }
        )
        result = response.json()
        print(result["response"])

asyncio.run(analyze_sports())
```
</CodeGroup>

## Architecture

BlitzAgent is built with modern, reliable technologies:

- **[Pydantic AI](https://ai.pydantic.dev/)**: Agent-centric design with full type safety
- **[FastAPI](https://fastapi.tiangolo.com/)**: High-performance async web framework
- **[Claude 4 Sonnet](https://anthropic.com/)**: Advanced reasoning capabilities with 200K context
- **[MCP Protocol](https://modelcontextprotocol.io/)**: Seamless tool integration for data access

<Note>
  BlitzAgent requires an Anthropic API key and access to the Blitz MCP server. See our [quickstart guide](/quickstart) for setup instructions.
</Note>
