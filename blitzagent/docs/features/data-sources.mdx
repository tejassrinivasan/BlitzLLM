# Data Sources

BlitzAgent leverages multiple comprehensive data sources to provide accurate, up-to-date sports analysis. Our three-tier data architecture ensures both historical depth and real-time accuracy across NBA and MLB analysis.

## Data Source Architecture

<Steps>
  <Step title="Historical Database (Primary)">
    **PostgreSQL databases** containing decades of comprehensive sports statistics
    
    **Priority**: Highest for statistical analysis and historical trends
    
    **Update Frequency**: Daily batch updates during season, end-of-season comprehensive updates
  </Step>
  
  <Step title="Web Scraping (Secondary)">
    **Real-time web data** from official sources and sports news sites
    
    **Priority**: Medium for current events, injury reports, and breaking news
    
    **Update Frequency**: Continuous updates throughout the day
  </Step>
  
  <Step title="Betting Markets (Tertiary)">
    **Live betting data** from major sportsbooks and odds providers
    
    **Priority**: Lower for market insights and predictive analysis
    
    **Update Frequency**: Real-time updates as markets move
  </Step>
</Steps>

## Historical Database Sources

### NBA Database

<AccordionGroup>
  <Accordion icon="basketball" title="Player Statistics Tables">
    **Coverage**: 1946-present (79 seasons)
    
    **Traditional Stats**:
    - `player_stats_regular_season`: PPG, APG, RPG, FG%, 3P%, FT%
    - `player_stats_playoffs`: Postseason performance metrics
    - `player_stats_advanced`: PER, BPM, VORP, Win Shares, Usage Rate
    
    **Game-Level Data**:
    - `game_logs`: Individual game performances
    - `shot_charts`: Spatial shooting data with coordinates
    - `plus_minus`: On-court impact metrics
    
    **Example Query**:
    ```sql
    SELECT player_name, season, ppg, apg, rpg, fg_pct, ts_pct
    FROM player_stats_regular_season 
    WHERE player_name = 'LeBron James' 
    ORDER BY season DESC;
    ```
  </Accordion>
  
  <Accordion icon="users" title="Team Performance Tables">
    **Coverage**: Complete franchise history for all 30 teams
    
    **Team Stats**:
    - `team_stats_regular_season`: Offensive/defensive ratings, pace, efficiency
    - `team_stats_playoffs`: Postseason team performance
    - `team_four_factors`: eFG%, TOV%, OREB%, FT Rate
    
    **Advanced Metrics**:
    - `team_lineup_stats`: Lineup combinations and effectiveness
    - `team_clutch_stats`: Performance in close games
    - `team_home_away_splits`: Venue-specific performance
    
    **Example Query**:
    ```sql
    SELECT team_name, season, wins, losses, off_rating, def_rating, net_rating
    FROM team_stats_regular_season 
    WHERE season = '2023-24' 
    ORDER BY net_rating DESC;
    ```
  </Accordion>
  
  <Accordion icon="calendar" title="Game and Schedule Data">
    **Coverage**: Every NBA game since 1946
    
    **Game Results**:
    - `games`: Scores, dates, venues, attendance
    - `game_box_scores`: Detailed team and player statistics
    - `game_play_by_play`: Shot-by-shot game progression
    
    **Schedule Information**:
    - `schedule`: Upcoming games, TV coverage, venue details
    - `injury_reports`: Player availability and injury status
    - `transactions`: Trades, signings, releases
  </Accordion>
  
  <Accordion icon="trophy" title="Historical Context">
    **Awards and Recognition**:
    - `awards_mvp`: MVP voting and winners
    - `awards_all_star`: All-Star selections and voting
    - `awards_all_nba`: All-NBA team selections
    
    **Draft and Development**:
    - `draft_history`: Draft picks, trades, and selections
    - `rookie_stats`: First-year player performance
    - `career_progression`: Player development over time
  </Accordion>
</AccordionGroup>

### MLB Database

<AccordionGroup>
  <Accordion icon="baseball" title="Pitching Statistics">
    **Coverage**: 1871-present (153 seasons)
    
    **Traditional Pitching**:
    - `pitching_stats`: ERA, WHIP, W-L, innings pitched, strikeouts
    - `pitching_advanced`: xFIP, SIERA, K/9, BB/9, HR/9
    - `pitching_situational`: Performance vs left/right, RISP, high leverage
    
    **Game-Level Pitching**:
    - `pitching_game_logs`: Start-by-start performance
    - `pitch_by_pitch`: Individual pitch data with velocity and location
    - `bullpen_usage`: Relief appearance patterns and effectiveness
    
    **Example Query**:
    ```sql
    SELECT pitcher_name, season, era, whip, strikeouts, innings_pitched
    FROM pitching_stats 
    WHERE season = 2024 AND innings_pitched > 100
    ORDER BY era ASC;
    ```
  </Accordion>
  
  <Accordion icon="chart-bar" title="Batting Statistics">
    **Coverage**: Complete offensive statistics for all players
    
    **Traditional Batting**:
    - `batting_stats`: AVG, OBP, SLG, OPS, HR, RBI, stolen bases
    - `batting_advanced`: wOBA, wRC+, ISO, BABIP, sprint speed
    - `batting_situational`: RISP, clutch, vs LHP/RHP
    
    **Plate Discipline**:
    - `plate_discipline`: Walk rate, strikeout rate, swing metrics
    - `batting_game_logs`: Game-by-game offensive performance
    - `hit_trajectory`: Launch angle, exit velocity, hard-hit rate
  </Accordion>
  
  <Accordion icon="shield" title="Defensive Metrics">
    **Coverage**: Fielding statistics and advanced defensive metrics
    
    **Traditional Defense**:
    - `fielding_stats`: Errors, assists, putouts, fielding percentage
    - `defensive_advanced`: DRS, UZR, OAA (Outs Above Average)
    - `catcher_defense`: Framing, blocking, throwing metrics
    
    **Positional Analysis**:
    - Position-specific metrics for each defensive role
    - Range and conversion rate statistics
    - Shift effectiveness and positioning data
  </Accordion>
  
  <Accordion icon="calendar" title="Game Results and Context">
    **Coverage**: Every MLB game with detailed context
    
    **Game Data**:
    - `games`: Scores, weather, attendance, ballpark factors
    - `box_scores`: Complete game statistics for teams and players
    - `play_by_play`: Inning-by-inning game progression
    
    **Contextual Factors**:
    - Weather conditions and their impact on play
    - Ballpark dimensions and offensive/pitching factors
    - Umpire tendencies and strike zone analysis
  </Accordion>
</AccordionGroup>

## API Integration Examples

<CodeGroup>
```python Data Source Query
# Query specific data sources
result = await client.analyze(
    "What data sources show LeBron's recent performance?",
    "Include database stats, recent news, and betting props"
)

# Response will detail:
# - Database: Season averages, advanced metrics
# - Web scraping: Recent injury reports, news
# - Betting: Current prop bet lines
```

```sql Direct Database Access
-- Example of historical database query
SELECT 
    p.player_name,
    p.season,
    p.ppg,
    p.apg,
    p.rpg,
    a.per,
    a.ts_pct
FROM player_stats_regular_season p
JOIN player_stats_advanced a ON p.player_id = a.player_id 
    AND p.season = a.season
WHERE p.player_name = 'Stephen Curry'
    AND p.season >= '2020-21'
ORDER BY p.season DESC;
```
</CodeGroup>
