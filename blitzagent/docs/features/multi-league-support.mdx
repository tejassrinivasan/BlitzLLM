## Automatic Detection

Blitz automatically detects NBA queries through:

<AccordionGroup>
  <Accordion icon="basketball" title="NBA Recognition">
    **Team Names**: Lakers, Warriors, Celtics, Bulls, Heat, etc.
    
    **Player Names**: LeBron James, Stephen Curry, Kevin Durant, etc.
    
    **Basketball Terms**: three-pointer, assists, rebounds, blocks, PER, usage rate, etc.
    
    **Example Queries**:
    - "How many three-pointers did Steph Curry make last game?"
    - "Lakers vs Warriors prediction for tonight"
    - "Which team has the best defensive rating?"
  </Accordion>
  
  <Accordion icon="brain" title="Contextual Intelligence">
    **Seasonal Context**: Understands when NBA season is active
    
    **Ambiguous Queries**: Asks for clarification when league cannot be determined
    
    **Player Context**: Recognizes current and former NBA players automatically
  </Accordion>
</AccordionGroup>

## Example Queries

<CodeGroup>
```bash Player Analysis
curl -X POST "https://blitzagent.onrender.com/analyze" \
  -H "Authorization: Bearer YOUR_BLITZ_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "query": "How many three-pointers did Stephen Curry make this season?"
  }'
```

```bash Team Comparison
curl -X POST "https://blitzagent.onrender.com/analyze" \
  -H "Authorization: Bearer YOUR_BLITZ_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "query": "Compare Lakers and Warriors offensive efficiency this season"
  }'
```
</CodeGroup>
