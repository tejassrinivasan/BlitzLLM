# Multi-League Support

BlitzAgent provides seamless support for both NBA and MLB analysis through intelligent league detection and specialized data sources. The system automatically determines which league your query refers to and applies the appropriate analysis framework.

## Automatic League Detection

BlitzAgent uses advanced natural language processing to automatically detect which sport and league your query refers to:

<AccordionGroup>
  <Accordion icon="basketball" title="NBA Detection">
    **Team Names**: Lakers, Warriors, Celtics, Bulls, etc.
    
    **Player Names**: LeBron James, Stephen Curry, Kevin Durant, etc.
    
    **Basketball Terms**: three-pointer, assists, rebounds, blocks, steals, PER, usage rate, etc.
    
    **Example Queries**:
    - "How many three-pointers did Steph Curry make last game?"
    - "Lakers vs Warriors prediction for tonight"
    - "Which team has the best defensive rating?"
  </Accordion>
  
  <Accordion icon="baseball" title="MLB Detection">
    **Team Names**: Yankees, Dodgers, Red Sox, Giants, etc.
    
    **Player Names**: Aaron Judge, Shohei Ohtani, Mike Trout, etc.
    
    **Baseball Terms**: ERA, WHIP, RBI, home runs, strikeouts, batting average, etc.
    
    **Example Queries**:
    - "What's Gerrit Cole's ERA this season?"
    - "Yankees vs Red Sox pitching matchup analysis"
    - "Best closer in baseball right now"
  </Accordion>
  
  <Accordion icon="brain" title="Contextual Intelligence">
    **Seasonal Context**: Understands when NBA season vs MLB season is active
    
    **Multi-Sport Players**: Correctly handles athletes who play multiple sports
    
    **Ambiguous Queries**: Asks for clarification when league cannot be determined
    
    **Cross-League Comparisons**: Can analyze across sports when explicitly requested
  </Accordion>
</AccordionGroup>

## Database Schema Support

BlitzAgent connects to specialized databases for each league with comprehensive historical data:

### NBA Database Schema

<CodeGroup>
```sql Player Stats Tables
-- Current season stats
SELECT * FROM player_stats_2024 
WHERE player_name = 'LeBron James';

-- Advanced metrics
SELECT * FROM advanced_stats_2024 
WHERE team_abbr = 'LAL';

-- Game-by-game performance
SELECT * FROM game_logs_2024 
WHERE player_id = 'jamesle01'
ORDER BY game_date DESC;
```

```sql Team Performance Tables
-- Team efficiency metrics
SELECT * FROM team_stats_2024 
WHERE team_abbr = 'BOS';

-- Four factors analysis
SELECT * FROM team_four_factors_2024;

-- Lineup combinations
SELECT * FROM lineup_stats_2024 
WHERE minutes_played > 50;
```
</CodeGroup>

### MLB Database Schema

<CodeGroup>
```sql Pitching Tables
-- Pitcher statistics
SELECT * FROM pitching_stats_2024 
WHERE player_name = 'Gerrit Cole';

-- Advanced pitching metrics
SELECT * FROM pitching_advanced_2024 
WHERE innings_pitched > 50;

-- Game-by-game starts
SELECT * FROM pitching_game_logs_2024 
WHERE pitcher_id = 'colege01';
```

```sql Batting Tables
-- Hitter statistics
SELECT * FROM batting_stats_2024 
WHERE player_name = 'Aaron Judge';

-- Situational hitting
SELECT * FROM batting_situational_2024 
WHERE situation = 'RISP';

-- Game-by-game performance
SELECT * FROM batting_game_logs_2024 
WHERE batter_id = 'judgaa01';
```
</CodeGroup>

## League Detection Examples

<CodeGroup>
```bash Clear NBA Query
curl -X POST "http://localhost:8001/analyze" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "query": "How many three-pointers did Stephen Curry make this season?"
  }'

# Response includes: "Detected league: NBA"
```

```bash Clear MLB Query
curl -X POST "http://localhost:8001/analyze" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "query": "What is Jacob deGrom'\''s ERA and strikeout rate?"
  }'

# Response includes: "Detected league: MLB"
```

```bash Ambiguous Query
curl -X POST "http://localhost:8001/analyze" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "query": "Who are the best teams this year?"
  }'

# BlitzAgent will ask for clarification or infer from context
```

```bash Multi-League Query
curl -X POST "http://localhost:8001/analyze" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "query": "Compare LeBron James'\'' dominance in NBA to Mike Trout'\''s in MLB",
    "extra_context": "Focus on how each player stands out relative to their peers"
  }'

# BlitzAgent handles both leagues in the analysis
```
</CodeGroup>

## Best Practices for Multi-League Queries

<AccordionGroup>
  <Accordion icon="target" title="Specific League References">
    **Good**: "NBA MVP candidates this season"
    
    **Better**: "Who are the top NBA MVP candidates this season based on advanced metrics?"
    
    Include sport-specific terminology to ensure accurate detection.
  </Accordion>
  
  <Accordion icon="calendar" title="Seasonal Context">
    **Important**: Remember that NBA and MLB seasons run on different schedules
    
    - NBA: October-June (with playoffs through June)
    - MLB: March-October (with playoffs through November)
    
    BlitzAgent understands seasonal context and will analyze accordingly.
  </Accordion>
  
  <Accordion icon="chart-bar" title="Metric Specificity">
    **NBA-specific**: PER, BPM, VORP, TS%, usage rate
    
    **MLB-specific**: ERA, WHIP, OPS, WAR, BABIP
    
    **Universal**: Games, wins, losses, performance trends
  </Accordion>
</AccordionGroup>
