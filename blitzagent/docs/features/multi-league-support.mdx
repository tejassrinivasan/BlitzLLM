## NBA Analysis Capabilities

<AccordionGroup>
  <Accordion icon="calculator" title="Statistical Analysis">
    From basic stats to advanced multi-conditional queries with unlimited filters and combinations.
    
    **Traditional Statistics:**
    - **Scoring**: PPG, FG%, 2PT%, 3PT%, FT%, TS%, eFG%
    - **Playmaking**: APG, AST%, assist-to-turnover ratio
    - **Rebounding**: RPG, ORB%, DRB%, total rebound percentage
    - **Defense**: SPG, BPG, steal percentage, block percentage

    **Advanced Metrics:**
    - **Impact metrics**: BPM, VORP, Win Shares, RPM
    - **Shooting analytics**: Shot distance, action area mapping, efficiency by zone
    - **Pace analysis**: Possessions per game, pace factor adjustments
    - **Lineup analytics**: Plus/minus by combination, minutes together
    
    <RequestExample>
    ```bash Advanced Statistical Query
    curl -X POST "https://blitzagent.onrender.com/analyze" \
      -H "Authorization: Bearer YOUR_BLITZ_API_KEY" \
      -H "Content-Type: application/json" \
      -d '{
        "query": "What are Jayson Tatum shooting percentages, usage rate, and efficiency metrics this season?",
        "extra_context": "Include FG%, 3P%, FT%, eFG%, TS%, PER, and compare to league averages"
      }'
    ```
    </RequestExample>
  </Accordion>

## Automatic League Detection

Blitz intelligently detects NBA queries through multiple contextual signals:

<AccordionGroup>
  <Accordion icon="basketball" title="Entity Recognition">
    **Team Detection:**
    - **Current teams**: Lakers, Warriors, Celtics, Bulls, Heat, Nets, etc.
    - **Historical teams**: SuperSonics, Bullets, Hornets (Charlotte/New Orleans)
    - **Franchise tracking**: Team relocations and name changes
    
    **Player Recognition:**
    - **Current players**: LeBron James, Stephen Curry, Kevin Durant, Giannis Antetokounmpo
    - **Historical legends**: Michael Jordan, Magic Johnson, Larry Bird, Kareem Abdul-Jabbar
    - **All-time database**: Every player from BAA (1946) through current NBA
    
    **Basketball Terminology:**
    - **Basic stats**: points, rebounds, assists, steals, blocks, field goal percentage
    - **Advanced metrics**: PER, usage rate, true shooting, effective field goal percentage
    - **Situational terms**: clutch time, playoff performance, home/away splits
  </Accordion>
  
  <Accordion icon="brain" title="Contextual Intelligence">
    **Seasonal Awareness:**
    - **NBA calendar**: Regular season (October-April), playoffs (April-June)
    - **Historical context**: Recognizes past seasons and specific date ranges
    - **Current events**: Injury reports, trades, roster changes
    
    **Query Disambiguation:**
    - **Ambiguous players**: Asks for clarification when multiple players share names
    - **Context clues**: Uses surrounding information to determine intent
    - **Follow-up questions**: Requests additional details when needed
    
    **Smart Defaults:**
    - **Current season**: Assumes current year unless specified otherwise
    - **Regular season**: Defaults to regular season stats unless playoffs specified
    - **Recent performance**: Prioritizes recent games for current analysis
  </Accordion>
</AccordionGroup>
