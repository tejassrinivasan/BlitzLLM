# Get Started with BlitzAgent

<img
  src="/images/blitzbot.png"

This guide will help you install, configure, and start using BlitzAgent for sports analysis in just a few minutes.

## Prerequisites

Before you begin, make sure you have:

<Steps>
  <Step title="Python 3.9+">
    BlitzAgent requires Python 3.9 or higher. Check your version:
    ```bash
    python --version
    ```
  </Step>
  <Step title="Anthropic API Key">
    Get your API key from [Anthropic Console](https://console.anthropic.com/). You'll need this for Claude 4 Sonnet access.
  </Step>
  <Step title="Access to Blitz MCP Server">
    Ensure you have access to the Blitz MCP server for sports data connectivity.
  </Step>
</Steps>

## Installation

<AccordionGroup>
  <Accordion icon="download" title="Clone the Repository">
    Clone the BlitzAgent repository to your local machine:
    
    ```bash
    git clone https://github.com/tejassrinivasan/BlitzLLM.git
    cd BlitzLLM/blitzagent
    ```
  </Accordion>
  
  <Accordion icon="package" title="Install Dependencies">
    Install the required Python packages:
    
    ```bash
    pip install -r requirements.txt
    ```
    
    This installs:
    - `pydantic-ai-slim[mcp]` for agent framework with MCP support
    - `fastapi` for the REST API server
    - `anthropic` for Claude 4 Sonnet integration
    - And other required dependencies
  </Accordion>
</AccordionGroup>

## Configuration

<AccordionGroup>
  <Accordion icon="key" title="Set Up Environment Variables">
    Create a `.env` file or set environment variables directly:
    
    ```bash
    # Required: Anthropic API Key
    export ANTHROPIC_API_KEY="your_anthropic_api_key_here"
    
    # Required: API Authentication
    export API_KEY="your_secure_api_key"
    
    # Optional: Server Configuration
    export HOST="0.0.0.0"
    export PORT="8001"
    export LOG_LEVEL="INFO"
    ```
    
    <Warning>
      Both `ANTHROPIC_API_KEY` and `API_KEY` are required. The `API_KEY` is used to authenticate requests to your BlitzAgent instance.
    </Warning>
  </Accordion>
  
  <Accordion icon="cog" title="Verify Configuration">
    Test your configuration by running a quick validation:
    
    ```bash
    python -c "from config import Config; Config.validate(); print('✅ Configuration is valid!')"
    ```
  </Accordion>
</AccordionGroup>

## Running BlitzAgent

<AccordionGroup>
  <Accordion icon="rocket" title="Start the Server">
    Launch BlitzAgent with the following command:
    
    ```bash
    python main.py
    ```
    
    You should see output similar to:
    ```
    INFO:     Starting BlitzAgent server...
    INFO:     MCP server initialized successfully
    INFO:     Agent created with MCP tools
    INFO:     Server running on http://0.0.0.0:8001
    ```
  </Accordion>
  
  <Accordion icon="globe" title="Test the API">
    Open another terminal and test your installation:
    
    ```bash
    curl -X GET "http://localhost:8001/health"
    ```
    
    You should receive a health check response confirming the server is running.
  </Accordion>
</AccordionGroup>

## Your First Analysis

Now let's run your first sports analysis query:

<CodeGroup>
```bash curl
curl -X POST "http://localhost:8001/analyze" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "query": "What are Stephen Curry'\''s three-point shooting stats this season?",
    "extra_context": "Focus on recent performance trends"
  }'
```

```python Python
import httpx
import asyncio

async def test_analysis():
    async with httpx.AsyncClient() as client:
        response = await client.post(
            "http://localhost:8001/analyze",
            headers={"Authorization": "Bearer YOUR_API_KEY"},
            json={
                "query": "What are Stephen Curry's three-point shooting stats this season?",
                "extra_context": "Focus on recent performance trends"
            }
        )
        
        if response.status_code == 200:
            result = response.json()
            print("✅ Analysis successful!")
            print(f"Response: {result['response'][:200]}...")
        else:
            print(f"❌ Error: {response.status_code}")
            print(response.text)

asyncio.run(test_analysis())
```

```javascript JavaScript
const response = await fetch('http://localhost:8001/analyze', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer YOUR_API_KEY'
  },
  body: JSON.stringify({
    query: "What are Stephen Curry's three-point shooting stats this season?",
    extra_context: "Focus on recent performance trends"
  })
});

const result = await response.json();
console.log('Analysis:', result.response);
```
</CodeGroup>

## Troubleshooting

<AccordionGroup>
  <Accordion icon="exclamation-triangle" title="Common Issues">
    **MCP Server Connection Failed**
    - Verify your internet connection
    - Check that the Blitz MCP server is accessible
    - Ensure proper database credentials in environment
    
    **Authentication Errors**
    - Verify your `ANTHROPIC_API_KEY` is valid
    - Ensure `API_KEY` is set for request authentication
    - Check that API keys are properly formatted
    
    **Import Errors**
    - Reinstall dependencies: `pip install -r requirements.txt`
    - Check Python version compatibility (3.9+)
  </Accordion>
  
  <Accordion icon="bug" title="Debug Mode">
    For detailed debugging information, set the log level to DEBUG:
    
    ```bash
    export LOG_LEVEL=DEBUG
    python main.py
    ```
    
    This will provide detailed logs about MCP connections, agent operations, and error details.
  </Accordion>
</AccordionGroup>
